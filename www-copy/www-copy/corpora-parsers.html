Программу можно использовать так:

  • $ mystem
    на входе - стандартный ввод, на выходе - стандартный вывод
  • $ mystem input
    на входе - файл input, на выходе - стандартный вывод
  • $ mystem - output
    на входе - стандартный ввод, на выходе - файл output
  • $ mystem input output
    на входе - файл input, на выходе - файл output

    Стандартный ввод и вывод, естественно, можно тоже перенаправлять <input >output

Опции


Опции надо указывать по правилам UNIX - до имен файлов, при этом можно склеивать, комбинировать и т.д.
  • -n построчный режим;
  • -c копировать весь ввод на вывод. То есть, не только слова, но и межсловные промежутки. Опция необходима для возврата к полному представлению текста. В случае построчного вывода (когда задана опция n) межсловные промежутки вытягиваются в одну строку, символы перевода строки заменяются на \r и/или \n. Пробел для большей видимости заменяется на подчеркивание. Символ \ заменятеся естественно на \\, подчеркивание на \_. Такми образом можно однозначно восстановить исходный текст;
  • -l не печатать исходные словоформы, только леммы и граммемы;
  • -i включает печать граматической информации расшифровка ниже;
  • -g склеивать информацию словоформ при одной лемме;
  • -r Только русские слова, игнорировать английские;
  • -s печатать марекер конца предложения. Только если -c задана;
  • -p сохранять регистр словоформы при приведении к лемме (title, lower, upper). Слова в начале предложения в title case преобразуются в lower;
  • -d Приписывать к лемме дизамбигуирующую информацию;
  • -x Рассматривать входный текст как xml - не парсировать элементы (<...>), комментарии (<!-- .. -->), CDATA, содержимое зоны noindex (<noindex> ... </noindex>), понимание entities (кириллические буквы а-я, ё, украинское е, ый, белорусское краткое у, десятеричное и, ять, ижица и фита, диакритика).
  • -b печатать только несловарные слова;

Расшифровка граммем


  • Часть речи

    • "A",Adjective,прилагательное
    • "ADV",Adverb,наречие
    • "CONJ",Conjunction,союз
    • "INTJ",Interjunction,междометие
    • "NUM",Numeral,числительное
    • "PART",Particle,частица
    • "PR",Preposition,предлог
    • "S",Substantive,существительное
    • "V",Verb,глагол

  • Время (глаголов)

    • "наст",Present,настоящее
    • "непрош",Notpast,непрошедшее
    • "прош",Past,прошедшее

  • Падеж

    • "им",Nominative,именительный
    • "род",Genetive,родительный
    • "дат",Dative,дательный
    • "вин",Ackusative,винительный
    • "твор",Instrumental,творительный
    • "пр",Ablative,предложный
    • "парт",Partitive,партитив (второй родительный)
    • "местн",Locative,местный (второй предложный)

  • Число

    • "ед",Singular,единственной
    • "мн",Plural,множественное

  • "Репрезентация" (наклонение) глагола

    • "деепр",Gerund,деепричастие
    • "инф",Infinitive,инфинитив
    • "прич",Participle,причастие
    • "изъяв",Indicative,изьявительное
    • "пов",Imperative,повелительное

  • Степень сравнения

    • "кр",Short,краткая
    • "прев",Superlative,превосходная
    • "срав",Comparative,сравнительная

  • Лицо глагола

    • "1-л",Person1,1-е лицо
    • "2-л",Person2,2-е лицо
    • "3-л",Person3,3-е лицо

  • Род

    • "жен",Feminine,женский
    • "муж",Masculine,мужской
    • "сред",Neuter,средний

  • Вид (аспект) глагола

    • "сов",Perfect,соврешенный
    • "несов",Imperfect,несовершенный

  • Залог

    • "страд",Passive,страдательный

  • Одушевленность (для существительных и винительного падежа прилагательных)

    • "од",Animated,одушевленное
    • "неод",Inanimated,неодушевленное
