Параметры семантического поиска

По умолчанию поиск производится по семантической разметке всех словарных значений слова. Например, если в поле "Семантические признаки" задан признак качества человека, то в корпусе будут найдены как прилагательные умный, верный, коварный, у которых этот признак характеризует основное значение слова, так и прилагательные мягкий и холодный, у которых значение человеческого качества - переносное.

Чтобы сузить область поиска, отметьте "галочкой" один или два параметра:

"1-е знач." - поиск по семантической разметке первого словарного значения слова (таким образом, вы признаку качества человека вы найдете примеры с прилагательными умный, верный, коварный, но не мягкий, холодный).

"др. знач." - поиск по семантической разметке прочих (переносных) значений слова (таким образом, по указанному признаку вы найдете только примеры с прилагательными мягкий, холодный и т.п.).

"фильтр1" - поиск в контекстах с разрешенной лексико-семантической омонимией, где слово выступает в первом значении. (Например, если задано прилагательное мягкий с семантическим признаком физические свойства, то будут найдены контексты типа мягкий снег или мягкая ткань, но не мягкий человек).

"фильтр2" - поиск в контекстах с разрешенной лексико-семантической омонимией, где слово выступает в других (переносных) значениях (таким образом, по запросу мягкий + качества человека вы найдете контексты типа мягкий человек или мягкий большевик).

Комбинация параметров "1-е знач." и "др. знач." задает поиск по всем значениям слова.

Комбинация параметров "фильтр1" и "фильтр2" задает поиск по всем контекстам со снятой лексико-семантической омонимией.
